### Prerequisites
- Node.js 18+
- Yarn Berry

### Installation

```bash
# Clone the repository
git clone https://github.com/samuelmholley1/dctbstudio.com.git
cd dctbstudio.com

# Install dependencies
yarn install

# Start development server
yarn dev
```
